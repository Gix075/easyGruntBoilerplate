simpleSASS
very simple SASS Boilerplate based on HTML5 Boilerplate

Overview

simpleSASS is a quite easy to use boilerplate for website styling setup.
Using this boilerplate you will ready to start a new SASS project in 5 minutes.

How to use

simpleSASS is just a simple HTML Boilerplate based on HTML5 Boilerplate.
This package will help you to start easy any new projects you want, and it can be used simply editing the HTML/JS and SASS files.
simpleSASS can be used also as a new Grunt project.
In order to customize the Grunt project, according your needs, you can customize the package.json and Gruntfile.js files.

To style your new project you can code the .scss files contained inside the "sass" project directory.
Remember that sass files need to be compiled using software as Compass, Grunt or other similar.
You can use each sass processor you want, but only Grunt is ready to use on this boilerplate.

index.html file

index.html is the html main file. It's only a demo file (that contains this documentation) that you can use as a new file model.

Using with Grunt

simpleSASS is ready to be used with Grunt (the javascript task runner) for SASS compiling. See more Here.
The Grunt Contribs used for this project are grunt-contrib-sass and grunt-contrib-watch but you can add all modueles you want and custmize the Gruntfile.js as you need.

Download package and open the main directory on your terminal. The simpleSASS folder will be your new project main dir.
Now be sure that you already have a local Grunt installation on your project. If not install it typing:
$ npm install grunt --save
When you have Grunt installed type the following command on you terminal in order to install the required Grunt modules:
$ npm install
Once Grunt modules are installed, you will ready to start coding and compiling SASS files simply typing "Grunt" on your console.
$ Grunt
Grunt default task will launch the first SASS compile and and the watch task that will autocompile all SASS files each time you will save .scss file.
